# JResources
Some resources I needed for my studies.

It can be imported as a JitPack dependency. 

<h2>WIth Gradle:</h2>

```
repositories {
  ...
  maven { url 'https://www.jitpack.io' }
}
```

and 

```
dependencies {
  implementation 'com.github.FosanzDev:JResources:-SNAPSHOT'
}
```
